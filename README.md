# Dice Roll Game
**Cocos Creator 3.8.3**

## 项目概述
这是一个基于 Cocos Creator 3.8.3 开发的骰子滚动游戏。玩家可以同时滚动多个骰子，观察结果，并根据特定规则计算分数。

## 已完成功能
- [x] 骰子滚动算法实现
  - [x] 使用物理系统模拟真实的骰子滚动效果
  - [x] 实现多个骰子同时滚动
  - [x] 检测骰子停止滚动的逻辑
